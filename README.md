# FxcnBeta

目前市场中的 cnBeta 官方客户端及其他第三方客户端的设计风格大体上未能向 Material Design 靠拢，于是就有了这个项目。

## release 配置

在本地 `local.properties` 中添加下列配置

```
WECHAT_APPID=
KEY_ALIAS=
KEY_PASSWORD=
STORE_FILE=
STORE_PASSWORD=
```
